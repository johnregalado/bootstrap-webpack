# Bootstrap Webpack.js

### Prerequisites
* Node.js

### Usage

#### download 

```shell
git clone https://github.com/johnregalado/bootstrap-webpack.git
```
#### install scripts
```shell
npm install
```

#### build scripts
```shell
webpack
```

#### run webpackserver
```shell
npm start
```
